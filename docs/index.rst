Welcome to Fire Toolbox's documentation!
===================================

**Fire Toolbox** is a collection of useful ADB (`Android Debug Bridge <https://developer.android.com/studio/command-line/adb/>`_) tweaks that can be applied to Amazon's Fire Tablets. 
The Toolbox project aims to help users fully customize and unlock the full potential of their tablets by putting all the power into their hands.

.. note::

   This project is under active development.
